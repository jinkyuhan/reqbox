# reqbox
cli tool for testing http API using cli tool quickly


```sh
# help

reqbox -h
reqbox config
reqbox req 
   --client (-c) curl | wget 
   --interactive (-i) boolean
reqbox req list
reqbox req last
reqbox req import 
   --raw  # Http Raw Format
   --curl # Curl commnd
   --json # Json notation
   --auto(default) # Try parse the above notations in order 
```
